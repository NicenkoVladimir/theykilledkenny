1. git clone url - clone git repository
2. git add file_name - указать файлы, которые нужно отслеживать
3. git commit -m "your message of changes" (локальный commit, без изменения на сервере)
4. git status - check status of local GIT repository
5. git push - выталкнуть на сервер

6. git pull - вытянуть данные из сервера на локальный ГИТ репозиторий
