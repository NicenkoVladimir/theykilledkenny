1. git clone url - clone git repository from server to your local (URL - HTTP or SSH)
2. git add file_name - указать файлы, которые нужно отслеживать (-A) - all files in directory
3. git commit -m "your message of changes" - локальный commit, без изменения на сервере, -ma 
4. git status - check status of local GIT repository
5. git push - upload commited files to server

6. git pull - download files from server to local repository
